# Tests API (REST Client)

Installe l’extension "REST Client" dans VS Code.

1) Lancer l’API : `npm run dev`
2) Ouvrir `http/health.http` puis cliquer "Send Request"
3) Pour un scénario complet : `http/commandes.http` et exécuter les requêtes dans l’ordre.
Cela sert à tester la base de donnée sans Front
